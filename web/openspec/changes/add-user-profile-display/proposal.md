## Why
当前用户登录后只能看到邮箱地址和退出登录选项，缺乏用户详细信息展示功能。用户无法查看自己的积分余额、账户状态等重要信息，影响用户体验和平台透明度。

## What Changes
- **在用户下拉菜单中添加"用户信息"选项**，点击后跳转到新的用户详情页面
- **创建独立的用户详情页面**，显示用户基本信息和积分信息
- **保持KISS设计原则**，使用简单直接的实现方式
- **高内聚低耦合**，新功能不影响现有认证和用户系统
- **充分测试覆盖**，确保功能稳定性和兼容性

**BREAKING**: 无破坏性变更，完全向后兼容

## Impact
- **受影响的规范**: user-profile, navigation, user-auth
- **受影响的代码**: HeaderBar.tsx组件、路由配置、新增UserProfilePage组件
- **数据库**: 复用现有用户表和积分表，无需新表
- **API**: 复用现有用户信息和积分查询API，无需新接口
