# Agent Trade 用户手册

## 1. 注册和登录

### 1.1 注册
要使用 Agent Trade，您首先需要注册一个账户：

1. 在 `http://localhost:3000` 或您的部署域名打开Web界面
2. 点击顶部导航栏中的"Register"（注册）按钮
3. 填写您的电子邮件地址和密码
4. 点击"Register"（注册）完成注册

### 1.2 登录
注册成功后，您可以登录：

1. 点击顶部导航栏中的"Login"（登录）按钮
2. 输入您注册时使用的电子邮件和密码
3. 点击"Login"（登录）访问您的仪表板

## 2. 配置

### 2.1 AI 模型 API 配置
要使用 AI 交易功能，您需要为您喜欢的 AI 模型配置 API 密钥：

1. 登录您的账户并导航到"AI Models"（AI模型）标签页
2. 选择您要使用的 AI 模型（DeepSeek、Qwen 等）
3. 输入您的 API 密钥和任何所需参数
4. 点击"Save"（保存）以应用配置

### 2.2 交易所配置
配置您的交易所 API 凭证：

1. 导航到"Exchanges"（交易所）标签页
2. 选择您喜欢的交易所（Binance、Hyperliquid、OKX）
3. 输入您的 API Key 和 Secret Key
4. 对于 OKX，还需要输入您的 Passphrase
5. 如果您想在没有真实资金的情况下测试，请启用"Testnet"（测试网）模式
6. 点击"Save"（保存）以应用配置

## 3. 交易者管理

### 3.1 创建新交易者
要创建 AI 交易者：

1. 导航到"My Traders"（我的交易者）标签页
2. 点击"Create Trader"（创建交易者）
3. 填写交易者名称，选择 AI 模型和交易所
4. 设置初始余额、杠杆和交易对
5. 如有需要，配置自定义提示
6. 点击"Create"（创建）保存交易者配置

### 3.2 启动/停止交易者
要控制您的交易者：

1. 在"My Traders"（我的交易者）页面上，找到您的交易者
2. 点击"Start"（启动）激活交易机器人
3. 点击"Stop"（停止）暂停交易
4. 在仪表板上监控实时状态和性能

### 3.3 优化交易者设置
提高交易者的表现：

1. 点击您要优化的交易者
2. 调整 AI 模型参数、杠杆或交易对
3. 更新自定义提示以完善交易策略
4. 保存更改并重新启动交易者

## 4. 风险管理

### 4.1 初始设置建议
- 开始测试时使用较小的初始余额
- 在真实交易前使用测试网模式
- 为初学者设置保守的杠杆（1-5x）

### 4.2 监控性能
- 定期检查"Statistics"（统计）和"Performance"（性能）标签页
- 分析交易决策并调整策略
- 设置止损和止盈规则
