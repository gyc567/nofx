## Why
首页导航栏的"User Manual"链接点击后跳转到了错误的URL "https://www.agentrade.xyz/#how-it-works"，而正确的URL应该是"https://www.agentrade.xyz/user-manual/en"或"https://www.agentrade.xyz/user-manual/zh"。

## What Changes
- 修复首页导航栏"User Manual"链接的跳转地址
- 确保点击链接后跳转到正确的用户手册页面
- 同时支持中英文版本
- 确保所有其他功能不受影响

## Impact
- Affected specs: `specs/docs/spec.md`
- Affected code: 前端的导航栏组件或用户手册链接组件
